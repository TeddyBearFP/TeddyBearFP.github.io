OpenCV 邊緣偵測

發表於 2018-01-08 | 分類於 OpenCV | 0 comments |

前言

有些圖片數據需要將多餘的值去除或是透明化，需要先把邊緣偵測出來，這樣才能掌握圖形

步驟

載入圖片(imread) -> 高斯模糊(GaussianBlur) -> 計算梯度(gradient) -> 找出邊緣後影像處理(imgProc) -> 存取圖片(imwrite)

高斯模糊

  • src：輸入可以為多通道圖，會單獨處理各通道，但是通常使用單通道灰階圖，例如CV_8U或CV_16U。
  • dst：輸出圖會和輸入圖尺寸、型態相同。
  • ksize：模板大小，長寬可以不同，但是都必須為正的奇數。
  • sigmaX：x方向的標準差。
  • sigmaY：y方向的標準差。
