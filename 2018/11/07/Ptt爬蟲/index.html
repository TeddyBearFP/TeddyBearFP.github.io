所謂靜態頁面是指純粹的HTML格式的頁面，這樣的頁面在瀏覽器中呈現的內容都在HTML原始碼中。

session

session，自動保存cookies，可以設置請求參數，下次請求自動帶上請求參數

requests

requests是一個很實用的Python HTTP客户端庫，編寫爬蟲和測試服務器響應數據時經常會用到。可以説，Requests 完全滿足如今網絡的需求

可以透過cmd輸入指令 pip3 install requests 安裝
